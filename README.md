# 拼多多风格"砍一刀"整蛊表白网页

一个充满土味和趣味的表白网页，模拟拼多多"砍价"玩法，让你的女朋友一边笑一边感动！

## 📝 项目简介

这是一个纯前端的单页面应用，采用拼多多的土味设计风格，通过7个搞笑步骤，最终以"恭喜中奖男朋友一枚"的方式表白。

### ✨ 核心亮点

- 🎯 **7个精心设计的步骤**
- 🔪 **20-30次"砍一刀"互动**（完美复刻拼多多的折磨感）
- 🎨 **土味十足的拼多多风格UI**
- 🎬 **丰富的动画效果**（闪烁、抖动、红包雨、礼花等）
- 🔊 **简单的音效系统**
- 📱 **响应式设计**（手机和电脑都能完美显示）

## 🎯 完整流程预览

1. **震撼开场**（5秒）
   - 显示"恭喜中奖"
   - 红包雨飘落
   - 提示需要砍价到0元

2. **新手教程**（3秒）
   - 介绍砍价玩法
   - 点击"我知道了"继续

3. **疯狂砍价**（2-4分钟）⭐ 核心环节
   - 第1-5刀：快速进展（每刀30-80元）
   - 第6-12刀：正常砍价（每刀10-30元）
   - 第13-17刀：进度变慢（每刀5-15元）
   - 第18-20刀：卡在99%（剩0.99元）
   - 第21-23刀：进度倒退！价格回弹
   - 第24-26刀：卡在99.99%（剩0.01元）
   - 第27-28刀：假失败（"网络错误"）
   - 第29-30刀：终于成功！爆炸特效

4. **假装加载**（5秒）
   - 显示"正在生成订单"等文字
   - 进度条动画

5. **躲避按钮**（10-20秒）
   - 鼠标靠近按钮会逃跑
   - 点击3次后才能点中

6. **红包炸开**（3秒）
   - 点击红包
   - 金币撒满屏幕

7. **最终表白**
   - 显示"男朋友"商品详情
   - 规格、功能、质保等信息
   - 点击"确认收货"完成表白

## 🎨 自定义修改

### 修改个人信息

打开 `index.html`，找到步骤7（第150行左右）：

```html
<div class="spec-value">180cm / 70kg（可自定义）</div>
```

修改为你的实际信息。

### 添加你的照片

1. 将你的照片放到 `assets` 文件夹
2. 在 `index.html` 中找到：

```html
<div class="placeholder-image">
    <span>👨</span>
    <p>（可替换成你的照片）</p>
</div>
```

替换为：

```html
<img src="assets/your-photo.jpg" style="width: 200px; height: 200px; border-radius: 15px;">
```

### 修改表白文案

在 `index.html` 的步骤7中，修改以下内容：
- 商品名称
- 商品详情
- 收货成功后的文字

### 调整砍价次数

在 `script.js` 中的 `handleBargain()` 函数中，修改各个阶段的条件判断数字即可。

### 修改配色

在 `styles.css` 中搜索颜色代码：
- `#FF4444` - 主红色
- `#FFD700` - 金色
- 修改为你喜欢的颜色
