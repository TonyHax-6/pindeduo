### 5. 技术细节

#### 字体回退机制
```css
font-family: 'Noto Sans SC', 'ZCOOL KuaiLe', 'Microsoft YaHei', 'PingFang SC', 'Hiragino Sans GB', 'SimHei', sans-serif;
```

即使在线字体加载失败，也会使用系统粗体字，配合CSS特效仍然很"土"。

#### 文字特效组合
1. **超粗字体**：font-weight: 900
2. **多层阴影**：2-4层立体阴影
3. **发光效果**：金色/彩色光晕
4. **描边效果**：-webkit-text-stroke
5. **字母间距**：letter-spacing 加大
6. **大字号**：关键元素字号增大

#### 配色方案
- 主色：鲜红 #FF0000, #FF4444
- 辅助：土豪金 #FFD700
- 发光：金黄 #FFA500
- 描边：深红 #8B0000, 黑色 #000

## 🔧 自定义调整

### 如果想调整字体土味程度

**更土一点**：
```css
/* 增加字重 */
font-weight: 900;

/* 增加发光强度 */
text-shadow: 0 0 30px #FFD700, 0 0 40px #FFA500;

/* 增加描边粗细 */
-webkit-text-stroke: 2px #000;
```

**稍微收敛一点**：
```css
/* 减少字重 */
font-weight: 700;

/* 减少阴影 */
text-shadow: 2px 2px 4px rgba(0,0,0,0.3);

/* 去掉描边 */
-webkit-text-stroke: 0;
```

### 如果想换其他字体

在 `index.html` 的 `<head>` 中修改字体链接，比如：

```html
<!-- 改用其他Google字体 -->
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@700;900&family=Ma+Shan+Zheng&display=swap" rel="stylesheet">
```

然后在 `styles.css` 中修改 `font-family`。
